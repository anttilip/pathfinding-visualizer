Pathfinding visualizer

This website visualizes pathfinding algorithms.

Currently available algorithms are Dijkstra, A* with several different heuristics, depth first and greedy best first.

A* is an extension to Dijkstra's algorithm and uses heuristics to determine which nodes should be visited next.

A* selects the path that minimizes f(n) = g(n) + h(n),
where n is the last node on the path, g(n) is the cost of path from the start node to n, and h(n) is a heuristic that estimates the cost of the cheapest path from n to the goal.

One of the most popular A* heuristic is Euclidean distance: 
// Euclidean distance
function(node, goal) {
    let dx = Math.abs(node.x - goal.x);
    let dy = Math.abs(node.y - goal.y);
    return Math.sqrt(dx * dx + dy * dy);
}
